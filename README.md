Cloud Computing and Big Data project

## Team Members

## Project Overview

In the context of this project, we want to evaluate how different Parquet compression codecs impact storage size, transfer time, and query time. To do so, we generated a dataset on financial transactions, which we we will use all along the project, applying on it the different codecs to perform our evaluation.
